# ⚡ 3 分钟快速开始 - MCP 项目批量处理

## 🎯 你只需要做 3 件事

### 第 1 步：找到你的 MCP 项目文件夹 📂

你昨天创建的所有 MCP 项目应该在一个文件夹里，比如：

```
E:\mcp-projects\          ← 这就是你的项目文件夹
├── weather-mcp\
├── translator-mcp\
├── database-mcp\
├── api-gateway-mcp\
└── notification-mcp\
```

**记下这个路径！**（例如：`E:\mcp-projects`）

---

### 第 2 步：修改配置 ✏️

1. 用记事本打开 `示例-批量处理MCP项目.bat`
2. 找到第 11 行：
   ```batch
   set "MCP_PROJECTS_DIR=E:\mcp-projects"
   ```
3. 把 `E:\mcp-projects` 改成你的实际路径
4. 保存并关闭

**就这么简单！**

---

### 第 3 步：运行脚本 🚀

双击 `示例-批量处理MCP项目.bat`

选择你想要的操作：

```
请选择操作:
======================================================================

  1. 📋 列出所有项目                    ← 先看看有哪些项目
  2. 🔧 为所有项目设置环境变量         ← 创建 .env 文件（含 API_KEY）
  3. 📦 批量安装依赖                    ← npm install / pip install
  4. 🔨 批量构建项目                    ← npm build / python build
  5. 🧪 批量运行测试                    ← 运行所有测试
  6. 🚀 批量发布项目                    ← 发布到 PyPI/NPM
  7. ⚡ 执行完整流程 (推荐)            ← 一键完成所有操作！
  8. 💡 自定义操作                      ← 高级用法
```

**推荐：直接选择 7！** 一次搞定所有事情 💪

## ❓ 遇到问题？

### Q: 找不到项目？

**A:** 检查文件夹路径是否正确：
```batch
set "MCP_PROJECTS_DIR=E:\mcp-projects"  ← 这里要改成你的实际路径！
```

### Q: 想修改 API Key？

**A:** 在 `示例-批量处理MCP项目.bat` 中修改第 14 行：
```batch
set "API_KEY=你的新key"
```

### Q: 某个项目失败了？

**A:** 没关系！脚本会继续处理其他项目。查看错误信息，单独修复该项目即可。

## 🚀 进阶用法

### 命令行模式（灵活）

```bash
# 查看所有项目
python batch_process_mcp.py "E:\mcp-projects"

# 完整流程
python batch_process_mcp.py "E:\mcp-projects" --action all

# 只处理项目 1, 3, 5
python batch_process_mcp.py "E:\mcp-projects" --action install --projects "1,3,5"

# 自定义 API Key
python batch_process_mcp.py "E:\mcp-projects" --api-key "your-key" --action env
```

### PowerShell 模式（Windows）

```powershell
.\batch_process_mcp.ps1 -ProjectsDir "E:\mcp-projects" -Action all
.\batch_process_mcp.ps1 -ProjectsDir "E:\mcp-projects" -Action install -Projects "1,3,5"
```
