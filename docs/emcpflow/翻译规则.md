# 翻译规则

## 特殊词汇翻译

### bachai / BACH / bachstudio

**正确翻译**：
- `bachai` → **巴赫** (Bach)
- `BACH` → **巴赫**
- `bachstudio` → **巴赫工作室** (Bach Studio)

**示例**：
```
包名: bachai-data-analysis-mcp
翻译: 巴赫数据分析服务器 ✅

包名: @bachstudio/mcp-file-search  
翻译: 巴赫工作室文件搜索服务 ✅

错误示例:
巴凯数据分析服务器 ❌
```
