# 更新日志

## Version 2.0 - 简化版发布 (2025-11-05)

### 🎯 重大更新

#### 新增简化版界面
- ✅ 极简操作：只需一个输入框
- ✅ 支持输入包地址或包名
- ✅ 自动识别 PyPI/NPM/Docker Hub
- ✅ 一键完成所有操作

#### 新增功能

**1. 包信息自动获取 (`package_fetcher.py`)**
- 从 PyPI API 获取 Python 包信息
- 从 NPM Registry 获取 Node.js 包信息
- 从 Docker Hub API 获取容器镜像信息
- 智能识别输入类型（URL 或包名）

**2. AI 模板生成 (`ai_generator.py`)**
- 集成 Azure OpenAI
- 自动生成专业的模板描述
- 支持中文描述生成
- 提供备用生成方案（无 AI 时）

**3. 配置管理增强 (`config_manager.py`)**
- 保存 EMCP 登录凭据
- 保存 Azure OpenAI 配置
- 自动保存 Session
- 配置文件持久化

**4. 自动登录**
- 配置一次凭据，后续自动登录
- Session 自动保存和恢复
- 无需每次手动输入

**5. 新增文件**
- `emcpflow_simple_gui.py` - 简化版主程序
- `package_fetcher.py` - 包信息提取器
- `ai_generator.py` - AI 模板生成器
- `build_simple.bat` - 简化版打包脚本
- `run_simple.bat` - 简化版运行脚本
- `QUICK_START_SIMPLE.md` - 快速开始指南
- `CONFIG_EXAMPLE.json` - 配置示例
- `使用说明.md` - 详细使用说明

**代码结构**

**错误处理**

### 📊 版本对比

| 功能 | v1.0 (旧版) | v2.0 (新版) |
|-----|-----------|-----------|
| **输入方式** | 本地文件夹 | 包地址/包名 |
| **登录方式** | 每次手动 | 自动登录 |
| **模板生成** | 手动填写 | AI 自动生成 |
| **包信息来源** | 本地文件 | 官方 API |
| **操作步骤** | 5+ 步 | 1 步 |
| **适用场景** | 本地项目 | 已发布包 |

### 🎯 使用建议

**简化版（推荐用于已发布包）**
```bash
python emcpflow_simple_gui.py
```

**完整版（适合本地项目）**
```bash
python emcpflow_gui.py
```
